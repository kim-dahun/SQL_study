그룹웨어 ver1

필요 필드(DB)

- 회원가입 - 필요 인수 (ID, 비밀번호, 이름, 전화번호, 이메일)
- 로그인 - 필요 인수 (ID, 비밀번호, 승인여부)
- 게시글 작성 - 필요 인수(ID, 이름, 직급, 글제목, 글내용, 댓글, 사번, 부서 )
- 쪽지 보내기 - 필요 인수(ID, 이름, 직급, 부서, 글내용 )
- 일정관리 - 필요 인수(ID, 이름, 직급, 일정제목, 일정세부, 시작날짜, 마지막날짜)
- 사원명부 - 필요 인수(사원명, 사번, 부서, 직급)
- 관리자 페이지 - 필요 인수 ( 승인여부, 부서, 사번, 직급, ID )

주요 메서드

 회원가입

- 회원가입에 입력한 정보를 DB로 전달하는 메서드
- 회원가입 시 중복된 ID가 있는지 검증하는 메서드
- 회원가입 시 이메일에 @가 입력되었는지 확인하는 메서드
- 비밀번호에 소문자,숫자가 입력되어있는지 확인하는 메서드
- 회원 가입 성공 및 실패를 알리는 팝업창을 띄우는 메서드

 로그인

- 로그인 시 입력된 ID, PW와 일치하는 데이터가 있는지 검증하는 메서드
- ID, 이름, 이메일 등을 입력하여 PW를 찾는 메서드
- 이름, 이메일, 사번 등을 입력하여 ID를 찾는 메서드
- 로그인 성공 시 다음 프레임을 띄우는 메서드

 메인메뉴

1) 게시글 작성

- 게시글을 작성하는 창을 띄우는 메서드
- 작성한 게시글을 DB로 전달하는 메서드
- 게시글 창이 뜰 때마다 DB 데이터를 토대로 리스트를 갱신하는 메서드
- 게시글을 삭제하는 메서드
- 게시글을 수정하는 메서드
- 게시글에 댓글을 작성하는 메서드
- 게시글에 댓글이 달리면, 해당 ID로 알림을 주는 메서드

2) 쪽지 보내기

- 쪽지 리스트에 쪽지를 띄우는 메서드
- 쪽지 리스트를 갱신하는 메서드
- 읽음, 읽지 않음 여부 표시하는 메서드

3) 일정 관리

- 직원들의 한달 치 스케줄 표시하는 메서드
- 일정을 등록하는 메서드
- 일정을 수정하는 메서드
- 일정을 삭제하는 메서드
- 일정을 보는 단위를 연,월,일 등으로 조절하는 메서드

4) 사원 명부

- 전체 사원 리스트를 표시하는 메서드
- 조직도 표기
